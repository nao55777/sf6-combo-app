# sf6-combo-app (AIコンボ解析ツール)
React・Firebase・AIを用いた画像解析によるコンボ提案Webアプリを開発中
## 概要
ストリートファイター6のスクリーンショットを解析し、状況に応じた最適なコンボを提案するWebアプリです。 
## 開発の動機
自分自身がSF6をプレイする中で、特定の状況での最適コンボを即座に確認したいと考え、独学で学習中のReactとAI技術を組み合わせたものを開発中。
## 現在の進捗と実装機能
‐ 開発状況: 現在基盤構築が完了し、解析ロジック実装に取り組んでいます。
‐ 実装済み: Reactによるフロントエンド構築、Firebaseとの連携、テスト用画像の表示機能。
‐ 実装中（TODO）: Hugging Face APIを活用したキャラクターおよび状況の自動判定ロジック。
‐ 判定結果に基づいた最適コンボのデータベース連携と表示。
 ## このプロジェクトの目的
技術の習得（React/Firebase/API連携）を主眼に置いており、エラーを自力で解決しながら一歩ずつ機能を拡張していくプロセスを大切にしています。
